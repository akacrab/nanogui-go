NanoGUI.go¶

Full Screen

NanoGUI.go is apure golang implementation of NanoGUI. NanoGUI is a minimalistic GUI library for OpenGL.

NanoGUI.go can run on desktops and browsers (I didn’t test on other environemnt).

It uses the following components:

  • OpenGL/WebGL library
  • Cross platform glfw wrapper
  • NanoVGo
